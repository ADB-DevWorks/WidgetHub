# WidgetHub

WidgetHub is a modular personal utility app built with .NET MAUI Blazor Hybrid.  
Each feature is designed as a standalone “widget” accessible from a central launcher screen.

## Planned Widgets
- **Grocery List** — Add items, check them off, keep unfinished items
- **Todo List** — Simple task manager
- **Blood Pressure Log** — Track readings, view by date, prepare for doctor visits

## Goals
- Offline-first (SQLite local storage)
- Clean, modular architecture
- Future cloud sync for backup and restore
- Easy to extend with new widgets

## Tech Stack
- .NET MAUI
- Blazor Hybrid
- EF Core + SQLite
- C#
